# Club Organization Record #  

## Introduction ##  
The spreadsheet included in this file was developed for my club organization during my tenure as Vice President. I proposed the creation of this workbook to provide a faster and more efficient method for monitoring our budget and expenses. It enables real-time tracking of available funds, total expenses, running balances, and projected future expenses. By having accurate, up-to-date information without the need for manual calculations, we were able to improve decision-making, respond more quickly to financial needs, and take action based on data-driven insights.

### Expenses ###
